# installing files and starting server

1. In the command line interface, navigate to the project folder and type **npm install**.
2. Run the app by typing **npm start** in the terminal. Runs the app in the development mode. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## Rules on playing the game

1. Guess the word by clicking a displayed letter.
2. The number of pink squares indicates the length of the word.
3. If you click on a correct letter, the letter position is revealed.
4. If you click on an incorrect letter, a part of the hangman is drawn.
5. If you guess all the letters correctly you win
6. After 10 incorrect guesses you lose.
